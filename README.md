#  Firebase Authentication

##  Descrição
Aplicação web simples utilizando Firebase Authentication para cadastro, login e controle de sessão de usuários.

## Funcionalidades
- Cadastro com email e senha
- Login de usuário
- Persistência de sessão
- Logout
- Interface dinâmica baseada no estado do usuário

##  Tecnologias
- HTML
- CSS
- JavaScript
- Firebase Authentication
